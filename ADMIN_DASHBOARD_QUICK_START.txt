                              SETUP (3 STEPS)
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Install Recharts (Charts Library)
─────────────────────────────────────────────────────────────────────────────
cd frontend
npm install recharts

STEP 2: Set Admin User
─────────────────────────────────────────────────────────────────────────────
python backend/set_admin.py bryantech.dev@gmail.com

Note: User must exist first. If not, sign up with that email first.

STEP 3: Restart Backend
─────────────────────────────────────────────────────────────────────────────
Stop current backend (Ctrl+C)
Run: python -m uvicorn app.main:app --reload

═══════════════════════════════════════════════════════════════════════════════
                          UPDATING PROFILE
═══════════════════════════════════════════════════════════════════════════════

1. Go to Settings
   Click "⚙️ Settings" in navbar or go to /admin/settings

2. Update Profile
   • Edit Full Name
   • Edit Bio (max 500 chars)
   • Add Avatar URL
   • Click "Save Profile"

3. Change Password
   • Click "Change Password"
   • Enter current password
   • Enter new password (min 8 chars)
   • Confirm new password
   • Click "Change Password"

                          TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Q: Charts not displaying?
A: Install recharts: npm install recharts

Q: Admin dashboard not accessible?
A: Run: python backend/set_admin.py bryantech.dev@gmail.com

Q: Password change fails?
A: Make sure current password is correct and new password is 8+ chars

Q: Profile update fails?
A: Check avatar URL is valid and bio is under 500 chars
