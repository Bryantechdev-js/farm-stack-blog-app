                          TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Q: Not redirecting to admin dashboard?
A: 1. Check user role in database (should be "admin")
   2. Run: python backend/set_admin.py bryantech.dev@gmail.com
   3. Restart backend
   4. Clear browser cache
   5. Login again

Q: Charts not displaying?
A: 1. Install recharts: npm install recharts
   2. Restart frontend
   3. Clear browser cache

Q: "Access Denied" on admin dashboard?
A: 1. User is not admin
   2. Run: python backend/set_admin.py bryantech.dev@gmail.com
   3. Restart backend
   4. Login again

Q: Backend not running?
A: 1. Stop current backend (Ctrl+C)
   2. Run: python -m uvicorn app.main:app --reload

Q: Frontend not running?
A: 1. Stop current frontend (Ctrl+C)
   2. Run: npm run dev (in frontend directory)
