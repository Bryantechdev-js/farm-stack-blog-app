                          SETUP (3 COMMANDS)
═══════════════════════════════════════════════════════════════════════════════

1. Install Charts Library
   cd frontend && npm install recharts

2. Set Admin User
   python backend/set_admin.py bryantech.dev@gmail.com

3. Restart Backend
   python -m uvicorn app.main:app --reload

                          TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Q: Charts not showing?
A: npm install recharts

Q: Not redirecting to admin dashboard?
A: python backend/set_admin.py bryantech.dev@gmail.com

Q: Admin dashboard not loading?
A: Restart backend and clear browser cache

Q: Password change fails?
A: Current password must be correct, new password 8+ chars
