================================================================================
                        HOW CORS WORKS
================================================================================

1. Frontend makes request to backend
2. Browser sends OPTIONS preflight request first
3. Backend responds with CORS headers
4. Browser checks if origin is allowed
5. If allowed, browser sends actual request
6. Backend responds with data

Before Fix:
  Auth middleware blocked OPTIONS requests
  CORS headers never sent
  Browser blocked request

After Fix:
  Auth middleware allows OPTIONS requests
  CORS middleware sends headers
  Browser allows request
  Data loads successfully

  - http://127.0.0.1:3000 (frontend IP)
  - http://localhost:8000 (backend dev)

  - Allow all headers

  - Expose all headers to frontend
