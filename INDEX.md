## 🚀 Quick Commands

```bash
# Backend
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000

# Frontend
cd frontend
npm install
npm run dev

# Docker
cd backend
docker-compose up

# MongoDB
mongosh "your-connection-string"
```
